# Notion Template: Strategy Scorecard

## Scorecard

```
┌──────────────────────────────────────┐
│  📋 SCORECARD          [STRATEGY]    │
├──────────────────────────────────────┤
│  Symbol: [___]    Date: [___]        │
│  Strategy: [___]   Time: [___]       │
├──────────────────────────────────────┤
│  SCORING                            │
│  ┌──────────────────────────────┐  │
│  │ Factor      │ Score │ Weight  │  │
│  ├──────────────────────────────┤  │
│  │ Momentum    │  _/10 │  40%   │  │
│  │ Fundamentals│  _/10 │  30%   │  │
│  │ Technical   │  _/10 │  20%   │  │
│  │ Sentiment   │  _/10 │  10%   │  │
│  ├──────────────────────────────┤  │
│  │ TOTAL SCORE │       │ ___/10  │  │
│  └──────────────────────────────┘  │
└──────────────────────────────────────┘
```

## Decision

```
○ BUY  (Score ≥ 8.0)
○ HOLD (Score 6.0-7.9)
○ SELL (Score < 6.0)
```

